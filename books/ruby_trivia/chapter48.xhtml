47. クラスメソッド定義

クラスやモジュールのメソッドを定義するときは、普通次のようにします。

class Calc
  class << self
    def >>(exp)
      eval exp
    end
  end
end

Calc >> '1 + 2' # => 3
Calc >> '10 ** 2' # => 100

外側のクラス定義をClass.newやModule.newで行えば、次のような書き方もできます。

class << Calc = Class.new
  def >>(exp)
    eval exp
  end
end

Calc >> '123 / 4.0' # => 30.75
Calc >> '2 * Math::PI' # => 6.283185307179586
